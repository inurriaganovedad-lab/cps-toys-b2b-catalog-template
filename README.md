# CPS Toys B2B Catalog Template

A lightweight, responsive front-end template for organizing a first wholesale toy sourcing brief. It demonstrates a practical buyer workflow instead of reproducing the supplier's production website.

## What it includes

- Category filtering for water-play, bubble, outdoor, and educational toys
- An inquiry shortlist that stays in the browser
- Quantity, market, packaging, and compliance planning fields
- Responsive layouts for desktop and mobile
- No build process, database, tracking, or form-data collection

## Run locally

Open `index.html` in a browser. The template uses the Lucide icon CDN and product imagery hosted by the supplier, so an internet connection is required for those assets.

## Buyer references

Use the template to prepare a brief, then verify current products and order details with the supplier:

- [CPS Toys wholesale toy supplier](https://www.cps-toys.com/)
- [Browse the CPS Toys product catalog](https://www.cps-toys.com/products/)

Product availability, MOQ, packaging, certificates, pricing, and lead times must be confirmed directly with CPS Toys before placing an order.
